Системное ПО — это набор программ, которые управляют компонентами компьютера, такими как процессор, коммуникационные и периферийные устройства, и создают среду для работы других программ.

Прикладное ПО — это программы, предназначенные для решения конкретных практических задач пользователей. Они ориентированы на работу с информацией пользователя и предоставляют удобный интерфейс для взаимодействия.
